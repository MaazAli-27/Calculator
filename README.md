# Simple Calculator

A simple, web-based calculator built with **HTML**, **CSS**, and **JavaScript**. This calculator supports basic arithmetic operations, percentages, backspace, and decimal numbers, all in a user-friendly interface.

---

## Features

- **Basic Operations**: Addition (+), Subtraction (-), Multiplication (×), Division (÷)
- **Percentage (%)**
- **Backspace (⌫)** – remove the last character
- **Clear (AC)** – reset the display
- **Decimal Support** – use "." for decimal numbers
- **Responsive Layout** – centered on the screen with a clean, modern design
- **Hover Effects** – buttons highlight when hovered
